# Should I Buy This?

A quick decision-making app that helps you figure out whether to make a purchase. Enter an item, price, income, and how badly you want it, and get an honest verdict, Buy It, Wait a Week, or Skip It, powered by GPT-5.6.

Built for OpenAI Build Week (Apps for Your Life track) with Codex.

**Live demo:** https://nurana100.github.io/should-I-buy-this-/

## What it does

You enter an item, its price, your monthly income, how badly you want it (1-10), and when you last bought something similar. The app runs those through a scoring formula and returns a verdict along with a short, honest explanation, either generated live by GPT-5.6 (if you paste in your own OpenAI API key) or by an offline fallback so the app always works instantly with no setup.

## How I collaborated with Codex

I'm a beginner, so I leaned on Codex heavily throughout this project rather than writing the app from scratch myself.

- **Scaffolding:** Codex generated the initial HTML structure, the form, and the styling for the receipt-style layout.
- **Decision logic:** I worked with Codex to design and refine the scoring formula that turns price, income, desire, and recency into a verdict.
- **GPT-5.6 integration:** Codex helped me wire in a live call to GPT-5.6 so the reasoning behind each verdict can be generated by the model itself, while keeping a safe offline fallback in case no API key is provided.
- **My decisions:** I chose the product idea, the three verdict categories (Buy It / Wait a Week / Skip It), the receipt visual concept, and made the call to keep API keys client-side and session-only rather than hardcoded, for security. Codex implemented these decisions and helped me debug issues along the way, like the live API call failing silently and the offline fallback not showing.

## Tech

Single-file HTML, CSS, and vanilla JavaScript. No build step, no dependencies. Optional live reasoning via the OpenAI API (GPT-5.6), entered client-side per session.

## Try it

Open the live link above, no login or install needed. Leave the API key field blank to use the offline scoring logic, or paste in your own OpenAI API key to get live GPT-5.6 reasoning.
