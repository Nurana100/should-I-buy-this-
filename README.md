# Should I Buy This?

A quick decision-making app that helps you figure out whether to make a purchase. Enter an item, price, income, and how badly you want it, and get an honest verdict, Buy It, Wait a Week, or Skip It, powered by GPT-5.6.

Built for OpenAI Build Week (Apps for Your Life track) with Codex.

**Live demo:** https://nurana100.github.io/should-I-buy-this-/

## What it does

You enter an item, its price, your monthly income, how badly you want it (1-10), and when you last bought something similar. The app runs those through a scoring formula and returns a verdict along with a short, honest explanation, either generated live by GPT-5.6 (if you paste in your own OpenAI API key) or by an offline fallback so the app always works instantly with no setup.


## Try it

Open the live link above, no login or install needed. Leave the API key field blank to use the offline scoring logic, or paste in your own OpenAI API key to get live GPT-5.6 reasoning.
